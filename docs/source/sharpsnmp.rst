#SNMP
=====

Core Library
------------

#SNMP Library implements a robust SNMP engine entirely in C#. It provides full
support for SNMPv1, SNMPv2c, and SNMPv3.

Key Features:
^^^^^^^^^^^^^

- **Full SNMPv1, v2c, and v3 Support**: Compliance with SNMP standards and
  main-stream vendor implementations.
- **IPv6 Support**: Fully supports managing IPv6 devices.
- **Asynchronous Operation**: Designed to handle large networks efficiently
  through asynchronous operations.
- **Trustworthy**: Used by many of Fortune 500 companies.

Getting Started:
^^^^^^^^^^^^^^^^

- Get-started guide and other articles are in `#SNMP Library Docs <https://docs.sharpsnmp.com/>`_.
- Source code and examples are hosted in `#SNMP GitHub repository`_.

Tools and Utilities
-------------------

#SNMP has a variety of tools and utilities designed to help manage and
monitor SNMP-enabled devices. These sibling projects complement the #SNMP
Library.

- `Command-line SNMP Tools and Test Agent <https://github.com/lextudio/sharpsnmplib-samples>`_: SNMP utilities for the command line and test agent sample.
- `#SNMP Pro <https://pro.sharpsnmp.com/>`_: Compile MIB files into C# and other MIB related tools.
